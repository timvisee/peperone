# peperone

A stupidly simple command-line in-the-background stopwatch.

```bash
$ peperone new mytimer

$ peperone show mytimer
00:00

$ peperone list
mytimer

# -- 21 seconds later --

$ peperone show mytimer
00:21

$ peperone remove mytimer
```
